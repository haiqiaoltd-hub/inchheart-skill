# 视频

这里指普通视频：教程、课程、网络收藏、创作素材、临时剪辑。不要套电影刮削结构。

| 场景 | 物理主轴 | 示例 |
|---|---|---|
| 创作素材 | 项目 | `Projects/影片名/Footage/` |
| 教程/课程 | 系列/来源 | `Videos/Courses/课程名/` |
| 网络收藏 | 来源/频道 | `Videos/Channels/频道名/` |
| 临时剪辑 | 工作流状态 | `Videos/待整理/`、`Videos/回收/` |

创作视频结构：

```
Projects/
└── 项目名/
    ├── Footage/
    ├── Audio/
    ├── Stills/
    ├── Exports/
    └── Refs/
```

规则：

- 明确服务某个项目时，项目永远优先于主题。
- 教程按课程/系列排序，不按“技术/艺术/哲学”混建。
- 网络视频按频道/来源，主题关系交给标签或笔记。
- 临时视频必须有定期清空的待整理区。
