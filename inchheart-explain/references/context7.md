# Context7

## What It Does

Context7 is a documentation-focused MCP and CLI for library/framework docs lookup.

## When to Use

- Current or version-specific API questions
- Library usage examples
- "What does the latest docs say?" type questions

## Split

- CLI can be used directly for docs lookup and setup.
- MCP is for agent integrations.

## Config

- In this machine, it is wired as an HTTP MCP server at `https://mcp.context7.com/mcp`.
- The key is stored in config, not in the skill.
