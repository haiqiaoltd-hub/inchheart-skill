# Local CLI Overview

## Common Tools

- `rg`: text search
- `fzf`: fuzzy selection
- `jq` / `yq` / `duckdb`: structured data work
- `sg` / `ast-grep`: code search
- `codegraph`: code graph and symbol analysis
- `firecrawl`: web search/scrape/crawl
- `defuddle`: clean article extraction
- `obscura`: rendered read-only scraping
- `agent-browser`: interactive browser automation
- `lit`: local PDF/Office parsing

## Practical Rule

- Use the narrowest tool that answers the question.
- Use `inchheart-search` for web and browser workflows.
- Use `inchheart-source` for source archive and CodeGraph maintenance.
