# 06 人物时间线

| 时间 | 事件 | 意义 |
|---|---|---|
| 1821 | 生于莫斯科 | 俄国宗教和城市贫困成为精神底色 |
| 1846 | 《穷人》发表 | 文学生涯开端 |
| 1849 | 被捕并经历模拟处决 | 死亡、罪责、救赎经验成型 |
| 1850-1854 | 西伯利亚苦役 | 与罪犯共同生活，重构人性观 |
| 1864 | 《地下室手记》 | 现代自我分裂表达成熟 |
| 1866 | 《罪与罚》 | 观念犯罪和忏悔主题成熟 |
| 1868-1869 | 《白痴》 | 纯善与社会败坏的冲突 |
| 1871-1872 | 《群魔》 | 政治虚无主义批判 |
| 1879-1880 | 《卡拉马佐夫兄弟》 | 晚期综合作品 |
| 1881 | 去世 | 影响现代小说、宗教哲学和存在主义 |
