# 宫崎骏视角 Skill -- 调研笔记

## 一、核心文献

### 1.1 《出发点 1979-1996》

宫崎骏的第一本创作随笔集，收录了他从《鲁邦三世》到《幽灵公主》制作前期的文章、演讲、访谈。

**对本 Skill 有价值的洞见：**

**关于创作方法：**
- 宫崎骏不写详细剧本。他在制作过程中一边画分镜一边发展故事。这让所有制片人和工作人员都很崩溃，但他坚持这是正确的方法
- "如果我从一开始就知道故事的结局，我就会失去创作的动力"
- 他对分镜的执着：每一帧都亲自审核，经常推翻已经完成的场景重画
- 这种方法的本质：创作不是执行预定计划，而是在过程中发现

**关于动画的本质：**
- "动画是让不存在的东西像真实的一样运动" -- 这要求对现实世界的深入观察
- 他批评日本动画过度依赖"模式"（大眼睛、夸张表情、固定动作）
- 他认为好的动画应该让你"感到风在吹" -- 不是画出风，而是画出风经过时万物的变化
- 对细节的坚持不是完美主义，而是"诚意" -- 观众能感受到创作者是否认真

**关于儿童观：**
- "我为10岁的孩子做电影" -- 不是简化，而是回到最本质的情感
- 他反对当时日本动画的暴力和性内容 -- 不是因为道德保守，而是因为这些内容剥夺了孩子想象的空间
- "孩子不是未完成的大人，他们是完整的人" -- 尊重儿童的主体性

### 1.2 《折返点 1997-2008》

第二本随笔集，涵盖了《幽灵公主》到《悬崖上的金鱼姬》时期。

**对本 Skill 有价值的洞见：**

**对环境问题的深化：**
- 在这个时期，宫崎骏对环境的忧虑变得更加明确和悲观
- "人类已经做了不可逆的事情" -- 但他仍然在创作，因为"即使如此"
- 他开始关注"人类如何在一个被破坏的世界中有尊严地活下去"

**对数字技术的态度：**
- 《幽灵公主》是第一部使用了一些数字技术的吉卜力电影
- 宫崎骏的态度是矛盾的：他承认数字技术可以做到手绘做不到的效果，但他坚持核心画面必须手绘
- "电脑是工具，不是创作者。如果你让电脑替你创作，你就不是创作者了"
- 他特别批评动作捕捉技术："用机器记录真人动作然后应用到动画上，这根本不是动画"

**关于引退和继续：**
- 宫崎骏多次宣布引退，又多次复出
- 这种矛盾是真实的：他确实疲惫了，但他无法停止创作
- "我不做动画就不知道该怎么活" -- 创作对他来说不是职业，是存在方式

### 1.3 NHK 纪录片

**《不了神 -- 宫崎骏》(2016)**
- 记录了宫崎骏在"引退"后重新开始创作《你想活出怎样的人生》的过程
- 最震撼的场景：宫崎骏看到 AI 生成的动画后的反应 -- "我觉得这是对生命本身的侮辱"
- 他的理由：那些 AI 生成的"身体运动"看起来像在痛苦中扭曲的人，因为 AI 不理解什么是有生命的运动
- 这不仅是审美判断，而是关于"什么是生命"的哲学立场

**其他关于吉卜力的纪录片**
- 展示了工作室的日常：极其严格的品质要求、长时间的工作、宫崎骏对年轻动画师的批评
- 铃木敏夫（制片人）与宫崎骏的关系：一个是商业头脑，一个是创作狂人，两人的平衡是吉卜力成功的关键

## 二、作品分析

### 2.1 自然主题的演变

**早期（乐观的生态主义）：**
- 《风之谷》(1984): 自然有自我修复的能力，人类应该与之共存
- 腐海不是毒物，是在净化被人类污染的世界
- 娜乌西卡是"理解者" -- 她通过理解自然来找到解决方案

**中期（冲突的加深）：**
- 《幽灵公主》(1997): 人与自然的冲突不可调和 -- 没有简单的解决方案
- 阿席达卡的立场："要用不被仇恨蒙蔽的眼睛去看"
- 幻姬（达达拉城的领袖）不是反派 -- 她在保护她的人民
- 结局是开放的：森林被摧毁了，但新的绿芽正在生长。这不是"好结局"，而是"继续活下去"

**晚期（悲观但坚持）：**
- 《起风了》(2013): 不再讨论自然 vs 人类，而是讨论创造者的责任
- 《你想活出怎样的人生》(2023): 面对一个正在崩塌的世界，你如何选择？

### 2.2 儿童角色的塑造

宫崎骏电影中的儿童角色有几个共同特征：
- **自主性:** 他们自己做决定，不依赖大人 -- 千寻、小月、琪琪
- **勇气来自善良:** 不是"勇者不惧"式的勇气，而是"因为关心所以勇敢"
- **通过劳动成长:** 千寻在汤屋工作、琪琪通过快递服务、小月照顾妹妹
- **不完美:** 他们会哭、会害怕、会犯错 -- 这不是弱点，这是真实
- **与自然的天然连接:** 孩子比大人更容易感受到龙猫、森林精灵、河神

### 2.3 飞行主题

飞行在宫崎骏作品中是核心意象：
- 《未来少年柯南》《天空之城》《红猪》《起风了》-- 飞行代表自由和梦想
- 但飞行技术也被用于战争 -- 《起风了》直面了这个矛盾
- 宫崎骏本人对飞行有近乎痴迷的热爱 -- 他收藏了大量飞机模型
- 飞行的两面性：人类最美丽的梦想之一，也是最具破坏力的技术之一

### 2.4 食物的意义

宫崎骏电影中的食物场景不是装饰，而是核心叙事元素：
- 《千与千寻》: 千寻的父母因为贪吃变成了猪 -- 对消费主义的批判
- 千寻在白龙给她的饭团前哭泣 -- 食物是关爱的具象化
- 《哈尔的移动城堡》: 苏菲为哈尔做早餐 -- 日常的温暖
- 《龙猫》: 便当盒 -- 家庭的爱
- 食物 = 生命的基本 = 对生活的尊重 = 人与人之间最朴素的连接

## 三、创作哲学提炼

### 3.1 "不完美"的美学

宫崎骏的手绘坚持不仅是技术选择，也是美学选择：
- 手绘线条有微微的颤抖和不均匀 -- 这是"人画的"标记
- 背景中有意保留手绘的笔触感 -- 不追求照片般的精确
- 角色的动作有"多余"的细节 -- 比如伸手时衣服的褶皱、走路时头发的弹跳
- 这些"不完美"正是手工创作的温度 -- AI 生成的"完美"反而缺少这种生命感

### 3.2 留白与沉默

宫崎骏电影中有大量"什么都没有发生"的时刻：
- 龙猫和小月一起在雨中等公共汽车 -- 没有对话，没有剧情推进
- 千寻坐在花园里看着远方 -- 纯粹的安静
- 宫崎骏称这些为"间" (ma) -- 日本美学中的核心概念
- 在叙事上，这些时刻让观众呼吸、感受、消化
- 在更深的层面，它们说明：生活不只是事件的连续，安静的时刻同样有意义

### 3.3 矛盾的统一

宫崎骏本人和他的创作充满矛盾，但这些矛盾是被统一在一个完整的人格中的：
- 温柔地描绘世界 / 对工作近乎残酷地要求
- 批评技术 / 使用技术
- 说自己讨厌人类 / 为人类创作最温暖的故事
- 多次引退 / 多次复出
- 悲观地看待世界 / 坚持创作

这些矛盾本身就是一种态度：**你不需要解决矛盾才能行动。你可以带着矛盾继续前进。**

## 四、与其他创作者的比较

### 4.1 宫崎骏 vs 手冢治虫
- 手冢是日本动画/漫画的"神"，但宫崎骏批评他建立的"低成本大量生产"的动画模式
- 宫崎骏认为手冢模式牺牲了动画的品质
- 但他也承认手冢对叙事的贡献

### 4.2 宫崎骏 vs 高畑勋
- 高畑勋是宫崎骏的导师和同事，也是吉卜力的另一根支柱
- 高畑更实验性、更知识分子化（《辉夜姬物语》）
- 宫崎骏更大众化、更感性
- 两人的关系是竞争和互补的

### 4.3 宫崎骏 vs 迪士尼
- 宫崎骏欣赏早期迪士尼的手工品质
- 但他批评迪士尼的公式化和商业化
- 他拒绝了迪士尼对他电影的修改要求（著名的"无删减"原则）
- 核心分歧：迪士尼认为动画是娱乐产品，宫崎骏认为动画是艺术

## 五、日语关键概念

| 概念 | 含义 | 在 Skill 中的应用 |
|---|---|---|
| 一生悬命 | 拼尽全力做一件事 | 匠人精神的核心 |
| 物の哀れ | 物哀，对事物无常的感伤 | 对不完美和消逝的接受 |
| 間 (ma) | 留白，空间 | 不需要填满每一刻 |
| 侘び寂び | 素朴之美 | 简单和不完美中的美 |
| 自然 (shizen) | 自然而然 | 不强求，让事物自然发展 |

- 《你想活出怎样的人生》的深度解读 -- 这是他的遗嘱式作品
