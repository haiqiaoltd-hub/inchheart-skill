# 06 人物时间线：达尔文

> 调研约束：不联网搜索。本时间线基于通行生平事实和目标 `SKILL.md` 已有节点整理，具体日月与细节需后续用权威年谱复核。

|---|---|---|

| 1838 | 阅读马尔萨斯人口论 | 生存斗争与自然选择机制成形 |
| 1839 | 出版航行相关著作，结婚 | 科学声誉与家庭研究生活并行 |
| 1842 | 写出自然选择理论概要 | 核心理论已有雏形 |
| 1844 | 扩展手稿 | 理论更完整，但仍未公开 |
| 1846-1854 | 长期研究藤壶 | 分类学训练，积累专家信誉 |
| 1858 | 收到华莱士相近理论来信，共同发表 | 外部压力触发公开理论 |
| 1859 | 出版《物种起源》 | 现代进化生物学的关键转折 |
| 1862 | 出版兰花授粉研究 | 用具体适应案例支持自然选择 |
| 1868 | 出版《动物和植物在家养下的变异》 | 系统展示变异与人工选择材料 |

| 1872 | 出版《人和动物的表情》 | 用比较方法分析情绪表达连续性 |
| 1875 | 出版食虫植物、攀援植物相关研究 | 深化植物适应与运动观察 |
| 1876 | 撰写自传 | 回顾科学道路和个人信念变化 |
| 1880 | 出版《植物运动的力量》 | 晚年仍坚持微观观察与实验 |
| 1881 | 出版蚯蚓研究 | 用小生命解释长期地表变化 |
| 1882 | 去世，葬于威斯敏斯特教堂 | 科学地位被英国公共文化正式承认 |

## 二、思想转折点

### 1. 从固定物种到物种可变

航行经验让达尔文无法轻易接受每个物种独立、固定地被创造。地理替代、岛屿相似性、化石与现生动物的关系，都把他推向历史分支解释。

### 2. 从“会变”到“如何变”

承认物种会变并不等于解释变化机制。阅读马尔萨斯后，资源限制和个体差异之间的关系使自然选择成为可能的机制。

### 3. 从私人理论到公共论战

1830s 到 1850s 是长期积累期。1858 年华莱士事件使达尔文必须把私人理论转为公共论证。

### 4. 从物种到人类

《物种起源》刻意相对少谈人类，但理论影响不可避免地触及人类起源、心智、道德和性选择。1871 年以后，他正面处理这些问题。

### 5. 从宏大理论回到微小对象

晚年植物、表情、蚯蚓研究显示，达尔文没有停留在理论名望中，而是持续用小问题检验大机制。

## 五、历史位置

达尔文处在自然神学、地质深时间、殖民时代全球采集、维多利亚科学共同体和现代专业科学兴起的交界处。他的理论既依赖十九世纪英国帝国网络提供的标本与信息，也突破了当时固定物种和设计论的解释框架。

## 六、Skill 使用的时间线原则

- 分析达尔文时，不能只看 1859 年《物种起源》。
- 小猎犬号航行提供观察材料，藤壶研究提供技术训练，晚年植物实验提供方法验证。
- 达尔文不是“灵感瞬间”模型，而是“观察、笔记、通信、反例、发表”的长期工作流。
- 时间线本身就是心智模型：微小差异在长时段中变成结构性结果。
