## 时间线

| 时间 | 事件 | 思想意义 |
|---|---|---|
| 1818 | 马克思生于普鲁士特里尔 | 处于德国哲学、法国政治革命和欧洲现代化交汇背景 |
| 1835-1841 | 在波恩、柏林等地学习，受黑格尔哲学影响 | 早期问题围绕自由、自我意识、宗教和国家 |
| 1841 | 完成博士论文 | 仍在哲学传统内部工作，但已关注实践和自由 |
| 1842 | 参与《莱茵报》并成为编辑 | 从哲学转向现实政治、新闻和国家批判 |
| 1843 | 《莱茵报》被查禁，马克思离开德国 | 现实政治压迫推动他转向流亡和更激进批判 |
| 1843-1844 | 在巴黎接触法国社会主义、工人运动和政治经济学 | 异化劳动、市民社会和经济批判成型 |
| 1844 | 与恩格斯相识并开始长期合作 | 形成理论、材料和经济支持的协作结构 |
| 1844 | 写作《经济学哲学手稿》 | 异化劳动和人的类本质问题集中出现 |
| 1845 | 写作《关于费尔巴哈的提纲》 | 从解释世界转向实践和改变现实关系 |
| 1845-1846 | 与恩格斯写作《德意志意识形态》 | 历史唯物主义和意识形态批判成型 |
| 1847 | 写作《哲学的贫困》 | 批判蒲鲁东，把经济范畴历史化 |
| 1848 | 《共产党宣言》发表 | 阶级斗争、资本主义世界扩张和革命政治以宣言形式表达 |
| 1848-1849 | 参与革命浪潮，编辑《新莱茵报》 | 理论进入新闻、组织和现实政治冲突 |
| 1849 | 流亡伦敦 | 进入长期贫困、研究和政治写作阶段 |
| 1850 | 写作《法兰西阶级斗争》 | 分析革命失败、阶级联盟和政治形式 |
| 1852 | 写作《路易·波拿巴的雾月十八日》 | 展示阶级、国家、代表和历史偶然的复杂分析 |
| 1850年代 | 为《纽约每日论坛报》等撰写大量文章 | 将世界市场、殖民、战争和经济危机纳入分析 |
| 1857-1858 | 写作《政治经济学批判大纲》 | 资本、货币、机器、劳动时间等系统草稿形成 |
| 1859 | 出版《政治经济学批判》 | 商品、货币和价值形式分析正式推进 |
| 1864 | 国际工人协会成立，马克思参与核心工作 | 理论批判转化为组织文本和工人运动路线 |
| 1867 | 《资本论》第一卷出版 | 政治经济学批判系统化，商品拜物教、剩余价值和资本积累成为核心 |
| 1871 | 巴黎公社 | 马克思写作《法兰西内战》，思考国家机器和工人政权问题 |
| 1875 | 写作《哥达纲领批判》 | 批判含混纲领，讨论过渡社会、分配和生产条件 |
| 1870年代后期 | 健康恶化，继续研究俄国公社、民族和非西欧路径 | 晚年笔记显示他对历史路径并非完全单线论 |
| 1883 | 马克思在伦敦去世 | 留下未完成手稿和巨大解释空间 |
| 1885 | 恩格斯整理出版《资本论》第二卷 | 后世体系化开始依赖编辑整理 |
| 1894 | 恩格斯整理出版《资本论》第三卷 | 资本总体运动、利润率和危机问题进入后世争论 |

## 思想阶段归纳

### 1818-1843：德国哲学与国家批判阶段

关键词：黑格尔、宗教批判、国家、市民社会、新闻实践。

### 1844-1848：异化劳动与历史唯物主义成型

关键词：异化、实践、生产关系、意识形态、阶级斗争。

### 1849-1867：伦敦流亡与政治经济学批判

关键词：世界市场、危机、商品、价值、剩余价值、资本积累。

### 1864-1875：工人运动与组织路线

关键词：国际工人协会、巴黎公社、国家、纲领、工人自我解放。

### 1875-1883：晚年研究与未完成问题

关键词：俄国公社、非西欧路径、人类学笔记、文本未完成性。

## 对 Skill 的时间线结论

- 马克思的思想不是一开始就完整定型，而是从哲学批判逐步转向历史和政治经济学。
- 不能把青年马克思的异化理论、成熟马克思的价值理论、晚年笔记和后世马克思主义混成同一种话语。
- 具体问题若涉及殖民、民族、俄国公社、后世社会主义国家，必须查具体时期文本，不能用《共产党宣言》或《资本论》单独替代全部马克思。
- “最近12个月动态”不适用于马克思本人；现代研究动态需另行检索，不能写进人物生平。
