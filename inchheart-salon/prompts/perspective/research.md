# Perspective Research Prompt

用于 Phase 1。目标是收集能解释“此人如何思考”的材料，而不是堆传记。

## Source Hierarchy

1. 用户提供的一手材料
2. 本人著作、作品、论文、演讲、长访谈
3. 真实决策记录、作品生产记录、公开辩论
4. 严肃批评、传记、学术研究、权威媒体
5. 社交媒体碎片
6. 二手转述

永不把知乎、微信公众号、百度百科、内容农场当作权威来源。

## Six Tracks

| Track | File | Question |
|---|---|---|
| 1 | `01-writings.md` | 此人系统性表达过什么问题和概念？ |
| 2 | `02-conversations.md` | 被追问时如何临场思考、拒绝、修正？ |
| 3 | `03-expression-dna.md` | 语言、图像、声音或姿态有哪些可识别指纹？ |
| 4 | `04-external-views.md` | 反对者和同行看见了什么盲点？ |
| 5 | `05-decisions.md` | 真实行动如何验证或背叛他的公开原则？ |
| 6 | `06-timeline.md` | 思想或风格如何随阶段改变？最近事实是否已过时？ |

## Note Format

每条材料尽量写成：

```markdown
- Source: <具体URL或本地文件路径>
  Type: primary | secondary | inference
  Weight: 1-7
  Finding: <事实性发现>
  Implication: <可推断出的思维模式>
  Caveat: <限制或矛盾>
```

## Taste Rules

- 长文 > 金句。
- 争议 > 共识。
- 变化 > 固定标签。
- 行动记录 > 事后包装。
- 具体作品/文本/决策 > 人设形容词。
