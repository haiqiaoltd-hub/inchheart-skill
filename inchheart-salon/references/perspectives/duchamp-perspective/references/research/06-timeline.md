# 06 人物时间线

| 时间 | 事件 | 意义 |
|---|---|---|
| 1887 | 生于法国 | 进入欧洲现代主义语境 |
| 1912 | 《下楼梯的裸女》争议 | 与传统绘画观看方式冲突 |
| 1913 | 《自行车轮》 | 现成品逻辑开始 |
| 1915 | 移居纽约 | 进入纽约先锋艺术圈 |
| 1917 | 《泉》事件 | 艺术制度、署名和选择问题爆发 |
| 1923 | 《大玻璃》阶段性中止 | 未完成、笔记和延迟理解成为方法 |
| 1930s | 转向国际象棋和复制盒 | 退出、复制、档案化 |
| 1968 | 去世 | 影响观念艺术与制度批判 |
